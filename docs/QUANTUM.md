# Quantum Foundation

The quantum package implements analytical hydrogen orbitals, emission spectra,
Numerov radial solving, perturbation helpers, variational estimates, split-
operator TDSE evolution, H2 electronic-structure wrappers, and 3D orbital
visualization primitives.

Core validation gates include the hydrogen Bohr energies, Balmer and Lyman
reference lines, the 1s radial probability maximum at one Bohr radius, Numerov
ground-state recovery, H2 reference energies, and qualitative orbital shapes.
