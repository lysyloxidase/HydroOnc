# HydroOnc H2 Therapy

Phase 6 models molecular hydrogen as a research-grade anti-cancer adjuvant:

- selective ROS chemistry with explicit competing-scavenger caveats
- Nrf2/NF-kB/p53 signaling modulation summaries
- CD8+ T-cell exhaustion and rescue ODEs in acidic tumor microenvironments
- anti-PD-1/H2 synergy simulation
- clinical response probability estimates from multi-omic features

Install for development:

```bash
pip install -e "packages/h2_therapy[dev]"
```

Optional ML backend:

```bash
pip install -e "packages/h2_therapy[dev,ml]"
```
